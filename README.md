# about

Tic-tac-toe game tutorial build in React.

# technologies

- React

# instructions

1.  clone repository

    ```bash
    git clone https://github.com/ingriddorioschulze/tic-tac-toe-game
    cd tic-tac-toe-game
    ```

2.  install dependencies

    ```bash
    npm install
    ```

3.  start the application on your local machine

    ```bash
    npm start
    ```
